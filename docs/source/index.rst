Welcome to the official documentation for :code:`kleinlibc` - a collection of modules for embedded systems written in C. 
The source code is available on `Github <https://github.com/kristianklein/kleinlibc>`_.

See the :doc:`Introduction </introduction/introduction>` to learn more about the intended use of the library,
which types of modules it may contain and which design choices where made when developing the library.

For build and usage instructions, see the :doc:`Getting started </getting_started/getting_started>` section.
